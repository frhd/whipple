# whipple

## environment
Required environment variables. Set these in *.env* file.
```
REACT_APP_TOKBOX_API
REACT_APP_TOKBOX_SECRET
REACT_APP_DATABASE_URL
```

## development

```
// start the server
npm run server

// open new terminal
// start the client
npm start
```

### Simulate production

```
// build
npm build

// start server
npm run server:prod
```
